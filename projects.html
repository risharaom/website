GreenSight

A machine learning model dedicated to placing more trees in the bay area

Status: In Development

This project is a machine learning model focused on promoting sustainability and environmental equity by identifying areas in the Bay Area with low tree coverage. Using geospatial data and greenery percentages, the model analyzes neighborhoods and predicts the average greenery within a 0.1-mile radius of given coordinates. By highlighting zones with limited vegetation, the model helps prioritize where new trees should be planted. The goal is to support urban reforestation efforts, reduce carbon emissions, and create healthier, greener communities through data-driven insights.
